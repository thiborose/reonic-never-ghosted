# Reonic Research Debrief Presentation

Open `presentation/index.html` in a browser.

The deck uses Reveal.js from a CDN and custom white-background / black-text styling. Speaker notes are included for the title slide.

Suggested flow:

1. Use the first 5 slides to align on the product thesis.
2. Use the example slides to choose the main demo scenario.
3. Use the final slides to decide the 48-hour build slices.
