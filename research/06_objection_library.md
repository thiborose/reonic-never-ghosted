## Objection Routing Principles

- Clarify the offer before persuading.
- Ask for missing facts before making claims.
- Use a human action for trust repair.
- Use an in-person visit only when it unlocks a physical or high-value decision.
- Pause when another touch would add no value.
- Debrief every action so the objection model learns from real outcomes.
