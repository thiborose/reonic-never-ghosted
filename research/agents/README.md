# Agent Workspaces

Each folder belongs to one research agent.

Rules:

- Work only in the assigned folder.
- Do not overwrite other agents' files.
- Use `sources.md` for source records.
- Use `evidence.md` for extracted claims.
- Use `findings.md` for synthesis.
- Use `handoff.md` for final handoff.
- Use `open_questions.md` for unresolved questions.
