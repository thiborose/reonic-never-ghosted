# Source Record Template

Use one record per source.

```yaml
source_id:
title:
url_or_location:
source_type:
publisher_or_origin:
author_or_owner:
date_published_or_collected:
date_accessed:
geography:
market_context:
customer_segment:
funnel_stage:
sample_size:
methodology:
data_type:
primary_claims:
quantitative_metrics:
verbatim_customer_language:
limitations:
bias_risks:
confidence_rating: unknown
relevance_to_reonic:
recommended_use_in_poc:
transferability_to_germany:
```

Confidence values: `high`, `medium`, `low`, `unknown`.
